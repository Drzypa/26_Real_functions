periodic point


Let f:X→X be a function and fn its n-th iteration. A point x is called a periodic point of period n of f if it is a fixed point of fn. The least n for which x is a fixed point of fn is called prime period or least period.

If f is a function ℝ to ℝ or ℂ to ℂ then a periodic point x of prime period n is called hyperbolic if |(fn)′⁢(x)|≠1, attractive if |(fn)′⁢(x)|<1 and repelling if |(fn)′⁢(x)|>1.
