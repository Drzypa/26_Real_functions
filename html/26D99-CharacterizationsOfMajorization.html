characterizations of majorization


Let ℰn be the set of all n×n permutation matrices that exchange two components. Such matrices have the form

[⋱01⋱10⋱]

A matrix T is called a Pigou-Dalton transfer (PDT) if

T=α⁢I+(1-α)⁢E

for some α between 0 and 1, and E∈ℰn.

The following are equivalent

  1. 1.

    x is majorized (http://planetmath.org/Majorization) by y.

  2. 2.

    x=D⁢y for a doubly stochastic matrix D.

  3. 3.

    x=T1⁢T2⁢⋯⁢Tk⁢y for finitely many PDT T1,…,Tk.

  4. 4.

    ∑i=1nθ⁢(xi)≤∑i=1nθ⁢(yi) for all convex function θ.

  5. 5.

    x lies in the convex hull whose vertex set is

    {(yπ⁢(1),yπ⁢(2),…,yπ⁢(n)):π⁢ is a permutation of ⁢{1,…,n}}.
  6. 6.

    For any n non-negative real numbers a1,…,an,

    ∑πa1xπ⁢(1)⁢a2xπ⁢(2)⁢⋯⁢anxπ⁢(n)≤∑πa1yπ⁢(1)⁢a2yπ⁢(2)⁢⋯⁢anyπ⁢(n)

    where summation is taken over all permutations of {1,…,n}.

The equivalence of the above conditions are due to Hardy, Littlewood, Pólya, Birkhoff, von Neumann and Muirhead.
