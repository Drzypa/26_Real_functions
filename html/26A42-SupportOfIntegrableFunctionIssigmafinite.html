support of integrable function is σ-finite


Theroem - Let (X,ℬ,μ) be a measure space and f:X→ℂ a measurable function. If f is integrable, then the support of f is σ-finite (http://planetmath.org/SigmaFinite).

It follows easily from this result that any function in an Lp-space (http://planetmath.org/LpSpace), with 1≤p<∞, must have σ-finite support.

: Let A0:=[1,∞[, and for each n∈ℕ let An:=[1n+1,1n[. Since f is integrable, we must necessarily have μ⁢(|f|-1⁢(An))<∞ for each n∈ℕ∪{0}, because

μ⁢(|f|-1⁢(An))⋅1n+1≤∫|f|-1⁢(An)|f|⁢𝑑μ≤∫X|f|⁢𝑑μ<∞.

Since f and |f| have the same support, and the the support of the latter is supp⁢|f|=⋃n=0∞|f|-1⁢(An), it follows that the support of f is σ-finite. □
