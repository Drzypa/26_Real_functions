Darboux’s theorem (analysis)


Let f:[a,b]→ℝ be a real-valued continuous function on [a,b], which is differentiable on (a,b), differentiable from the right at a, and differentiable from the left at b. Then f′ the intermediate value theorem: for every t between f+′⁢(a) and f-′⁢(b), there is some x∈[a,b] such that f′⁢(x)=t.

Note that when f is continuously differentiable (f∈C1⁢([a,b])), this is trivially true by the intermediate value theorem. But even when f′ is not continuous, Darboux’s theorem places a severe restriction on what it can be.
