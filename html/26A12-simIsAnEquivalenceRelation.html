∼ is an equivalence relation


Note that ∼ as defined in the entry Landau notation is an equivalence relation on the set of all functions from ℝ+ to ℝ+. This set of functions will be denoted in this entry as F.

Reflexive (http://planetmath.org/Reflexive): For any f∈F, limx→∞⁡f⁢(x)f⁢(x)=1, and f∼f.

Symmetric: If f,g∈F with f∼g, then limx→∞⁡f⁢(x)g⁢(x)=1. Thus:

limx→∞⁡g⁢(x)f⁢(x)=limx→∞⁡1(f⁢(x)g⁢(x))=11=1

Therefore, g∼f.

Transitive (http://planetmath.org/Transitive3): If f,g,h∈F with f∼g and g∼h, then limx→∞⁡f⁢(x)g⁢(x)=1 and limx→∞⁡g⁢(x)h⁢(x)=1. Thus:

limx→∞⁡f⁢(x)h⁢(x)=limx→∞⁡(f⁢(x)g⁢(x)⋅g⁢(x)h⁢(x))=1⋅1=1

Therefore, f∼h.
