irrotational field


Suppose Ω is an open set in ℝ3, and 𝐕 is a vector field with differentiable real (or possibly complex) valued component functions. If ∇×𝐕=0, then 𝐕 is called an irrotional vector field, or curl free field.

If 𝐔 and 𝐕 are irrotational, then 𝐔×𝐕 is solenoidal.
