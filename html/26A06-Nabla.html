nabla


Let f:ℝn→ℝ be a C1⁢(ℝn) function, that is, a partially differentiable function in all its coordinates. The symbol ∇, named nabla, represents the gradient operator, whose action on f⁢(x1,x2,…,xn) is given by

∇⁡f = (fx1,fx2,…,fxn)
= (∂⁡f∂⁡x1,∂⁡f∂⁡x2,…,∂⁡f∂⁡xn)

Properties

  1. 1.

    If f,g are functions, then

    ∇⁡(f⁢g)=(∇⁡f)⁢g+f⁢∇⁡g.
  2. 2.

    For any scalars α and β and functions f and g,

    ∇⁡(α⁢f+β⁢g)=α⁢∇⁡f+β⁢∇⁡g.

The ∇ symbolism

Using the ∇ formalism, the divergence operator can be expressed as ∇⋅, the curl operator as ∇×, and the Laplacian operator as ∇2. To wit, for a given vector field

𝐀=Ax⁢𝐢+Ay⁢𝐣+Az⁢𝐤,

and a given function f we have

∇⋅𝐀 =∂⁡Ax∂⁡x+∂⁡Ay∂⁡y+∂⁡Az∂⁡z
∇×𝐀 =(∂⁡Az∂⁡y-∂⁡Ay∂⁡z)⁢𝐢+(∂⁡Ax∂⁡z-∂⁡Az∂⁡x)⁢𝐣+(∂⁡Ay∂⁡x-∂⁡Ax∂⁡y)⁢𝐤
∇2⁡f =∂2⁡f∂⁡x2+∂2⁡f∂⁡y2+∂2⁡f∂⁡z2.
