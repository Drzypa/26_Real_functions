derivatives of hyperbolic functions


In this entry we compute the derivative of the hyperbolic functions sinh⁡(x) and cosh⁡(x).

Recall that by definition:

sinh⁡(x) := ex-e-x2
cosh⁡(x) := ex+e-x2.

Therefore:

dd⁢x⁢sinh⁡(x) = dd⁢x⁢(ex-e-x2)
= 12⋅dd⁢x⁢(ex-e-x)
= 12⋅(ex-(-e-x))
= ex+e-x2
= cosh⁡(x).

Similarly dd⁢x⁢cosh⁡(x)=sinh⁡(x). Using the quotient rule, we compute the derivative of tanh⁡(x)=sinh⁡(x)cosh⁡(x):

dd⁢x⁢tanh⁡(x)=cosh2⁡(x)-sinh2⁡(x)cosh2⁡(x)=1cosh2⁡(x)

where we have used the equality cosh2⁡(x)-sinh2⁡(x)=1.
