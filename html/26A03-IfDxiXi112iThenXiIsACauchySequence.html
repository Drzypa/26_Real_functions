if d⁢(xi,xi+1)<1/2i then xi is a Cauchy sequence


Lemma 1.

Suppose x1,x2,…, is a sequence in a metric space. If for some N≥1, we have d⁢(ai,ai+1)<1/2i for all i≥N, then {xi} is a Cauchy sequence.

Proof.

Let us denote by d the metric function. If ε>0, then for some N∈ℕ we have 1/2N<ε. Thus, if N<m<n we have

d⁢(xm,xn) ≤ d⁢(xm,xm+1)+⋯+d⁢(xn-1,xn)
= (12)m+⋯+(12)n-1
= (12)m-1⁢∑i=1n-m(12)i
= (12)m-1⁢1-(12)n-m1-12
< (12)m
< (12)N
< ε,

where we have used the triangle inequality and the geometric sum formula (http://planetmath.org/GeometricSeries). ∎
