ostensibly discontinuous antiderivative


The real function

x↦15-3⁢cos⁡x (1)

is continuous for any x (the denominator is always positive) and therefore it has an antiderivative, defined for all x.  Using the universal trigonometric substitution

cos⁡x:=1-t21+t2,d⁢x=2⁢d⁢t1+t2,t=tan⁡x2,

we obtain

5-3⁢cos⁡x=5⁢(1+t2)-3⁢(1-t2)1+t2=2⁢(1+4⁢t2)1+t2,

whence

∫d⁢x5-3⁢cos⁡x=∫d⁢t1+4⁢t2=12⁢arctan⁡2⁢t+C=12⁢arctan⁡(2⁢tan⁡x2)+C.

This result is not defined in the odd multiples of π, and it seems that the function (1) does not have a continuous antiderivative.

However, one can check that the function

x↦x4+12⁢arctan⁡sin⁡x3-cos⁡x+C (2)

is everywhere continuous and has as its derivative the function (1); one has

|sin⁡x3-cos⁡x|≦13-1=12<π2.
