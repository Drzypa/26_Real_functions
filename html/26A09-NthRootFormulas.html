nth root formulas


In the following formulas, a is a nonnegative real number and other letters positive integers.  For other formulas, see the parent entry (http://planetmath.org/NthRoot).

  1. 1.

    0n= 0,   1n= 1

  2. 2.

    a1=a

  3. 3.

    anm=am⁢n=amn

  4. 4.

    am⁢kn⁢k=amn

  5. 5.

    amn=(an)m

  6. 6.

    am⋅an=am+nm⁢n

Proof.  For proving, one uses the definition of http://planetmath.org/node/754nth root and the power (http://planetmath.org/GeneralAssociativity) laws.

1∘.  0n=0,1n=1

2∘.  a1=a

3∘.  (anm)m⁢n=((anm)m)n=(an)n=a

4∘.  (amn)n⁢k=((amn)n)k=(am)k=am⁢k

5∘.  ((an)m)n=((an)n)m=am

6∘.  (am⋅an)m⁢n=(am)m⁢n⁢(an)m⁢n=((am)m)n⁢((an)n)m=an⁢am=am+n
