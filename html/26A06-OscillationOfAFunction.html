oscillation of a function


Definition 1.

Let f:X⊂R→R. The oscillation of the function f on the set X is said to be

ω⁢(f,X)=supa,b∈X⁡|f⁢(b)-f⁢(a)|,

where a,b are arbitrary points in X.

0.1 Examples

  • •

    ω⁢(x2,[-1,2])=4

  • •

    ω⁢(x,[-1,2])=3

  • •

    ω⁢(x,(-1,2))=3

  • •

    ω⁢(sgn⁡x⁢[-1,2])=2

  • •

    ω⁢(sgn⁡x⁢[0,2])=1

  • •

    ω⁢(sgn⁡x⁢(0,2])=0

Cauchy’s criterion can be expressed in terms of this concept.[1]
