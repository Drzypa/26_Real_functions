minimal and maximal number


Let’s consider a finite non-empty set  A={a1,…,an}   of real numbers or an infinite but compact (i.e. bounded and closed) set A of real numbers.  In both cases the set has a unique least number and a unique greatest number.

  • •

    The least number of the set is denoted by  min⁡{a1,…,an}  or  min⁡A.

  • •

    The greatest number of the set is denoted by  max⁡{a1,…,an}  or  max⁡A.

In both cases we have

min⁡A=inf⁡A,
max⁡A=sup⁡A,
min⁡A≦x≦max⁡A ∀x∈A,

where  inf⁡A  and  sup⁡A  are the infimum and supremum of the set A.

The min and max are set functions, i.e. they map subsets of a certain set to ℝ.

The min and max have the following distributive properties with respect to addition:

min⁡{a1,…,an}+b=min⁡{a1+b,…,an+b}
max⁡{a1,…,an}+b=max⁡{a1+b,…,an+b}

The minimal and maximal number of a set of two real numbers obey the formulae

min⁡{a,b}=a+b2-|a-b|2,
max⁡{a,b}=a+b2+|a-b|2,
max⁡{a,b}-min⁡{a,b}=|a-b|,
max⁡{a,b}+min⁡{a,b}=a+b,
max⁡{a,-a}=|a|
