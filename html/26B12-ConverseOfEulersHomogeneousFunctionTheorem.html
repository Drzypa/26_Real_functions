converse of Euler’s homogeneous function theorem


Theorem.  If the function f of the real variables x1,…,xk satisfies the identity

x1⁢∂⁡f∂⁡x1+…+xk⁢∂⁡f∂⁡xk=n⁢f, (1)

then f is a homogeneous function of degree n.

Proof.  Let  f⁢(t⁢x1,…,t⁢xk):=φ⁢(t).  Differentiating with respect to t we obtain

φ′⁢(t)=x1⁢fx1′⁢(t⁢x1,…,t⁢xk)+…+xk⁢fxk′⁢(t⁢x1,…,t⁢xk)=1t⁢[t⁢x1⁢fx1′⁢(t⁢x1,…,t⁢xk)+…+t⁢xk⁢fxk′⁢(t⁢x1,…,t⁢xk)],

which by (1) may be written

φ′⁢(t)=nt⁢f⁢(t⁢x1,…,t⁢xk)=nt⁢φ⁢(t).

Accordingly,

φ′⁢(t)φ⁢(t)=nt,

which implies the integrated form

ln⁡|φ⁢(t)|=ln⁡tn+ln⁡C

for any positive t.  Thus we have  φ⁢(t)=C⁢tn,  where C is independent on t.  Choosing  t=1  we see that  C=φ⁢(1),  and therefore  φ⁢(t)=tn⁢φ⁢(1).  This last equation means that

f⁢(t⁢x1,…,t⁢xk)=tn⁢f⁢(x1,…,xk)

saying that f is a (positively) homogeneous function of degree n.
